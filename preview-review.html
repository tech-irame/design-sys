Chart four-channel rule + table fallback
§18.1 · §18.2

Color + shape + text (value) + legend = four channels. <figure> with an aria-labelledby caption and a collapsible <details><summary>View as table</summary> fallback — real <table> for screen readers.

Risk heatmap — patterns under `prefers-contrast: more`
§18.4 · §17.8

Critical cells carry diagonal stripes, High gets cross-hatch, Medium gets dots — the color tells you the bucket, the pattern tells you the severity redundantly. The count text is the third channel.

Z-index scale — no more 9999 arms race
§17.7

Every z-index resolves to one of these tokens. Raw numeric values are a lint violation.

10 z-raised sticky page header, table filter
20 z-sticky pinned columns, bulk bar
30 z-overlay popover, dropdown, tooltip
50 z-modal dialog / drawer + scrim
60 z-toast toast stack
70 z-palette command palette · wins focus
Quick-reference — what §17 asks of every surface
§17
Always
  • Native element first — <button>, <a>, <label for>, <nav>, <main>, <table>
  • cursor: pointer on every clickable surface
  • aria-label on icon-only buttons; icon aria-hidden="true"
  • Skeleton for anything larger than a button
  • Skip link as the first focusable element
  • Shadcn FormField + FormMessage + Zod over 3 fields
  • Four-channel rule on every chart + <table> fallback
  • Z-index from the scale: 10 / 20 / 30 / 50 / 60 / 70
Never
  • Layout shift on hover (no width/height/padding transitions)
  • outline: none without a replacement
  • Raw z-index: 9999
  • <div role="button"> when <button> exists
  • Disabled submit to "hint" at validation — let it submit and land errors
  • Custom error text outside FormMessage (won't announce)
  • Color as the only channel on any chart
  • Placeholder-only form inputs (no associated label)

Connection state matrix
§13.6 · new

Six states every integration passes through. The pulse dot + word is the canonical visual — consistent across every card, row, and meta line in the app.

Connected
Connected
Link is live, last sync within tolerance. Green dot is static.
Syncing
Syncing
Active pull in flight. Blue dot pulses at 1.2s cycle (respects reduced-motion).
Idle
Idle
Connection works but last sync is outside the SLA window. Suggest "Sync now."
Error
Error
Sync failed — auth, network, permission. Card footer becomes "Reconnect →" in risk color.
Paused
Paused
Admin paused via audit-trail action. Carries "paused by · time" in meta.
Configuring
Configuring
Setup incomplete — awaiting credentials, SSO link, or approver. Footer shows "Resume setup →".
